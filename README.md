# pg_spatialid

Spatial ID (空間ID) utilities for PostgreSQL/PostGIS.

## Status

- Extension version: 0.1.0
- Implements Spatial ID (空間ID) specification v1.0

## Installation

```sh
make install
```

## Usage

```sql
CREATE EXTENSION pg_spatialid CASCADE;
```
