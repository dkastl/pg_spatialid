# pg_spatialid Extension Overview

`pg_spatialid` is a PostgreSQL extension that provides spatial identifier utilities and versioning information for Spatial ID (空間ID).

## Features

- Custom composite types for versioning
- Utility functions for extension and spec version info

## Usage

After installing the extension, you can use the provided types and functions in your SQL commands.
